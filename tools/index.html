BrowserSync Git
一、安装
可以全局安装或作为开发依赖安装

npm install browser-sync -g

npm install browser-sync --save-dev

二、静态网站
使用服务器模式，可以监听 css、html 文件

browser-sync start --server --files "**/*.html, **/*.css"

三、动态网站
使用代理模式

browser-sync start --proxy "主机名" "**/*.css"

其它方式

可以结合 gulp、grunt 使用
