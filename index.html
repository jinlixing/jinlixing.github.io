这是个人 GitHub 主页项目，使用 Bootstrap4 搭建而成，开发静态站时使用了 browser-sync 来辅助开发。browser-sync 主要用来进行浏览器同步测试，可在 PC、手机、平板 等环境下同步进行测试。

开发时使用服务器模式开启 browser-sync 即可：browser-sync start --server --files "**/*.html, **/*.css"
